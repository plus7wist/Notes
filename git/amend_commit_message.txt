++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
    修改提交的注释信息
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

经常有提交信息写错的情况。

如果写错的是最后一次提交，那么使用 git commit --amend 即可修改它。

如果写错的是之前某次你的提交：

git checkout <commit_hash>
git commit --amend
git checkout master
# 这时 git 会提醒你有个新的提交没有合并
git rebase <new_commit_hash>
