set textwidth=78
set colorcolumn=80
set formatoptions+=Bm

formatoption

    B join 行的时候，在多字节处不插入空格。

    m 允许多字节字符处换行。

    r 换行自动添加注释符号

    o 用 o O 插入新行的时候自动插入注释符号

format text

    gq 自动处理一个范围内的换行。

    :center :ce
        文字居中

    :left   :le
        文字左对齐，可以用作去缩进。

    :right  :ri
        文字右对齐。
