## 简介

从这个例子开始

```
let i = 1
while i < 5
    echo "i = " i
    let i = i + 1
endwhile
```

1. 赋值使用 let
2. echo 输出
3. 现代式脚本
4. 出现死循环，ctrl+c退出（windows下使用ctrl+break）。
5. let i = i + 1 可以简写成 let i += 1

例二

```
for i in range(1, 4)
    echo "i = " i
endfor
```

## 变量

1. 全局变量：用 g:varname 或 varname 表示，可以跨脚本。
2. 局部变量：用 s:varname 表示，仅脚本内部使用，注意，脚本结束不会使得变量销毁，下次还可以。
3. let 声明变量；unlet 取消变量的声明。

**exists()**函数，用来检查一个脚本是否用过。
